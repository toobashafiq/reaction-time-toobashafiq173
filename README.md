# ⌨️ Reaction Time Lab

A simple and interactive **Reaction Time Game** built using **HTML, CSS, and JavaScript**.
Test how fast you can respond to letters and numbers!

## 🚀 Features

* 🎯 Random stimulus (Letter or Number)
* ⏱️ Real-time reaction time tracking
* 📊 Stats panel:

  * Last Reaction Time
  * Total Trials
  * Correct Responses
  * Average Reaction Time
* 🔄 Reset functionality (Button + Space key)
* ⚡ Fast and responsive UI


## 🎮 How to Play

1. A random **letter or number** will appear on the screen.
2. Press:

   * **A** → if it's a **letter**
   * **L** → if it's a **number**
3. Your reaction time will be recorded.
4. Press **Space** or click **Reset** to restart.
